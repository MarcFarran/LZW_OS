## Instalación
```
Primero, debes elegir entre el User Kit y el Dev Kit. Se recomienda que los nuevos usuarios comiencen con el User Kit, pero luego pasen al Dev Kit.

El Dev Kit es la fuente activa contra la cual el equipo de Cosmos desarrolla directamente.
El Dev Kit tiene las características más recientes y avanzadas, pero en varios momentos presenta problemas conocidos y, a veces, ni siquiera puede compilarse.
Por lo tanto, para usar el Dev Kit, asegúrate de unirte a nuestros canales de soporte e informarte sobre el estado actual antes de usarlo o actualizarlo.

El User Kit es una versión estable y en forma de instantánea de Cosmos, que incluye un instalador preconfigurado.
Sin embargo, el User Kit suele estar algo desactualizado en comparación con el Dev Kit y solo se actualiza de manera ocasional.
El User Kit es una excelente manera fácil de familiarizarse con Cosmos,
pero los desarrolladores activos deben hacer la transición al Dev Kit después de haberse familiarizado bien con el UserKit,
 y estar preparados para encontrar algunos errores de vez en cuando.
```
